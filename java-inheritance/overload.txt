    trong abstract class có thể có

        abstract method -> các class con kế thừa bắt buộc phải Overide lại.

        method bình thường

abstract có thể có thuộc tính - interface không có thuộc tính

Trong interface các method là abstract.

Khi nao dung abstract, Khi nao dung interface
    nếu có thuộc tính [field] thì dùng abstract
    nếu không liên quan đến thuộc tính [field] thì dùng interface
